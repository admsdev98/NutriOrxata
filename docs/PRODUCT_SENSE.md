# Product Sense

## Objective

NutriOrxata replaces per-client spreadsheets with a tenant-scoped platform for planning, tracking, and communication.

## Primary Users

- Worker: nutritionist/trainer managing multiple clients.
- Client: end user following plans and logging progress.

## Core Value

- Fast client management workflows.
- Reusable content libraries.
- Clear weekly execution for worker and client.

## Success Indicators

- New client onboarding can be completed in under 10 minutes.
- Weekly plan assignment can be completed in under 5 minutes using templates.
- Client can identify today actions immediately on app open.

## Non-Goals (Current Scope)

- Full billing automation.
- Social feed behavior.
- Wearable integrations.
- Advanced medical EHR behavior.
