# Product Copy (MVP)

This document contains the exact copy for sensitive trust/access states.

Language:

- Default UI language is Spanish.
- Copy should be calm and non-aggressive.

## 1) Worker email not verified (blocked)

Title:

- "Verifica tu correo"

Body:

- "Te hemos enviado un enlace de verificacion. Abre tu correo y confirma para acceder a la aplicacion."

Actions:

- Primary: "Reenviar enlace"
- Secondary: "Cambiar email"

## 2) Trial expired (worker read-only)

Title:

- "Cuenta en modo lectura"

Body:

- "Tu periodo de prueba ha finalizado. Puedes consultar tus datos y exportarlos, pero no puedes realizar cambios hasta reactivar la cuenta."

Actions:

- Primary: "Exportar datos"
- Secondary: "Reactivar cuenta" (contact/support/manual unlock)
- Danger: "Borrar mis datos"

## 3) Tenant expired (client blocked)

Title:

- "Acceso no disponible"

Body:

- "Tu profesional tiene la cuenta inactiva en este momento. Contacta con el/ella para reactivar el acceso."

Actions:

- Primary: "Cerrar sesion"

## 4) Delete my data (tenant purge)

Warning (short):

- "Esta accion elimina tus datos de forma permanente en los sistemas activos."

Backup note:

- "Las copias de seguridad estan cifradas y se eliminan por rotacion en un plazo maximo de 30 dias."

Confirmation:

- "Escribe BORRAR para confirmar"
